## AI Resume ATS Analyzer

### Gemini-Powered Resume vs Job Description Matching Engine

This project is an AI-driven Applicant Tracking System (ATS) analyzer that evaluates how well a candidate’s resume matches a given Job Description. It uses Google’s Gemini Generative AI to provide:

- ATS match score
- Skill match analysis
- Missing keyword detection
- Section-wise feedback
- Resume improvement recommendations

The system is designed for job seekers, recruiters, and career coaches to assess and improve resume relevance before applying.
